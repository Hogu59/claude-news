## 主要な新機能と実践活用

### v2.1.149 — /usageカテゴリ別分析、/diffキーボード操作、セキュリティ修正 (5/22)

v2.1.149はコスト可視性とUI操作性を改善し、複数のセキュリティ問題を修正したアップデートです。

**`/usage`カテゴリ別コスト分析**: `/usage`が全体の使用量だけでなく、**skills、subagents、plugins、MCPサーバー別**のコスト内訳を表示するようになりました。どのコンポーネントが使用量を消費しているかが一目で分かり、プラグインやMCPサーバーのコスト効率を実測ベースで判断できます。

```bash
# カテゴリ別使用量を確認
/usage
# Skills: 12%, Subagents: 45%, Plugins: 8%, MCP: 35%
```

**`/diff`キーボードナビゲーション**: 詳細ビューで矢印キー、`j`/`k`、`PgUp`/`PgDn`、`Space`、`Home`/`End`キーでスクロールできるようになりました。大規模なdiffをマウスなしで素早く確認できます。

**GFMチェックボックスレンダリング**: マークダウンの`- [ ]`/`- [x]`タスクリストが通常の箇条書きではなく、**チェックボックスとしてレンダリング**されるようになりました。Claudeが生成するTODOリストや進捗報告がより視覚的に分かりやすくなりました。

**エンタープライズ設定 — `allowAllClaudeAiMcps`**: 新しい管理設定で、`managed-mcp.json`と併せてclaude.aiクラウドMCPコネクタをロードできます。エンタープライズ環境でマネージドMCPとクラウドMCPを並行利用できるようになりました。

[GitHub v2.1.149](https://github.com/anthropics/claude-code/releases/tag/v2.1.149)

## 開発者ワークフローティップス

### /usageカテゴリ分析 — 使用量消費の原因を追跡する

v2.1.149の`/usage`カテゴリ別分析は、**「なぜ使用量が早く消費されるのか」**という疑問に答えてくれます。Uberが4ヶ月で年間予算を使い切った事例が示すように、コスト可視性はAIコーディングツール運用の要です。

**実践的な活用パターン**:
1. **プラグインコスト監査**: コンテキストを過度に消費するプラグインを特定して無効化
2. **サブエージェント最適化**: 並列エージェント数とコストの相関を確認し、適切な水準に調整
3. **MCPサーバー比較**: 同機能のMCPサーバー間のコスト差を実測して効率的なサーバーを選択

特にv2.1.143で追加されたマーケットプレイスのトークンコスト表示と組み合わせると、**インストール前の予測コスト**と**実際の使用コスト**を比較してプラグインのROIを定量化できます。

[GitHub v2.1.149](https://github.com/anthropics/claude-code/releases/tag/v2.1.149)

## セキュリティ・制限事項

### v2.1.149セキュリティ修正 — PowerShell権限バイパス、worktreeサンドボックス (5/22)

v2.1.149で複数のセキュリティ関連バグが修正されました。

**PowerShell権限バイパス**: PowerShell組み込みの`cd`関数（`cd..`、`cd\`、`cd~`、`X:`）がClaude Codeの権限システムを**検知されずに作業ディレクトリを変更**できる問題が修正されました。

**worktreeサンドボックス範囲エラー**: git worktreeのサンドボックス書き込み許可リストが、共有`.git`ディレクトリだけでなく**リポジトリルート全体**をカバーするバグが修正されました。worktree内のエージェントが意図より広い範囲にファイルを書き込む可能性がありました。

**権限分析ギャップ**: `PWD`/`OLDPWD`/`DIRSTACK`環境変数の古い値による権限分析エラーが修正されました。

**推奨対応**: Windowsユーザーはv2.1.149への即時アップデートを推奨します。worktree分離機能（`isolation: "worktree"`）を使用しているチームは、以前のバージョンで作成されたworktreeの再作成を検討してください。

[GitHub v2.1.149](https://github.com/anthropics/claude-code/releases/tag/v2.1.149)
