## 주요 신규 기능 & 실전 활용

### v2.1.142 — Fast Mode Opus 4.7 기본 전환, Agent 실행 옵션 대폭 확장 (5/14)

v2.1.142는 Fast Mode의 모델 업그레이드와 에이전트 실행 제어를 강화하는 업데이트입니다.

**Fast Mode → Opus 4.7 기본**: `/fast` 모드가 기존 Opus 4.6에서 **Opus 4.7**로 기본 전환되었습니다. Opus 4.7은 고난이도 소프트웨어 엔지니어링 작업에서 4.6 대비 유의미한 성능 향상을 보이며, Fast Mode의 2.5배 빠른 출력 속도와 결합하면 복잡한 리팩토링이나 대규모 코드 생성에서 체감 차이가 큽니다.

**`claude agents` 플래그 확장**: 에이전트 실행 시 세밀한 제어가 가능해졌습니다.

```bash
# 특정 디렉토리에서 에이전트 시작, 모델·권한·MCP 설정 지정
claude agents --add-dir ./subproject \
  --model opus \
  --permission-mode plan \
  --mcp-config ./custom-mcp.json \
  --plugin-dir ./my-plugins \
  --effort high
```

새로 추가된 플래그: `--add-dir`, `--settings`, `--mcp-config`, `--plugin-dir`, `--permission-mode`, `--model`, `--effort`, `--dangerously-skip-permissions`

**SKILL.md 플러그인 인식**: 플러그인 루트에 `SKILL.md` 파일만 있고 `skills/` 디렉토리가 없는 경우에도 스킬로 자동 인식됩니다. 단일 스킬을 제공하는 간단한 플러그인의 구조가 단순해졌습니다.

**데몬 안정성 수정**: 바이너리 업그레이드 후 데몬이 깔끔하게 종료되지 않아 디스패치된 에이전트가 크래시루프에 빠지던 문제가 수정되었습니다. Windows 네트워크 드라이브에서 `claude agents` 데드락 문제도 해결되었습니다.

## 개발자 워크플로우 팁

### claude agents 플래그로 프로젝트별 에이전트 프리셋 구성하기

v2.1.142의 새 플래그를 활용하면 프로젝트마다 다른 에이전트 구성을 스크립트로 관리할 수 있습니다. 예를 들어 모노레포에서 서브프로젝트별로 다른 MCP 설정과 플러그인을 적용하거나, CI 환경에서 `--dangerously-skip-permissions`와 `--effort` 조합으로 자동화 파이프라인을 구성할 수 있습니다.

```bash
# package.json scripts에 프리셋 등록
# "agent:frontend": "claude agents --add-dir ./apps/web --mcp-config .mcp-frontend.json --model sonnet"
# "agent:backend": "claude agents --add-dir ./apps/api --mcp-config .mcp-backend.json --model opus --effort high"
```

핵심은 **MCP 설정과 플러그인을 용도별로 분리**하여, 에이전트가 불필요한 도구를 로드하지 않게 하는 것입니다. 이렇게 하면 컨텍스트 소비가 줄고 도구 선택의 정확도가 올라갑니다.

[GitHub v2.1.142](https://github.com/anthropics/claude-code/releases)

### Fast Mode Opus 4.7 — 언제 /fast를 쓰고 언제 끄는가

Fast Mode가 Opus 4.7로 업그레이드되면서 "항상 Fast Mode"가 합리적인 선택이 되었습니다. 다만 비용이 프리미엄이므로 사용 전략이 필요합니다.

**Fast Mode 적합**: 대규모 리팩토링, 보일러플레이트 생성, 테스트 작성, 문서화 — 속도가 품질에 영향을 주지 않는 작업

**일반 모드 적합**: 아키텍처 설계, 복잡한 버그 분석, 보안 코드 리뷰 — 깊은 추론이 필요한 작업

```bash
# Opus 4.7 Fast Mode 활성화 (v2.1.142부터 기본)
/fast

# 환경변수로 명시적 활성화 (이전 버전 호환)
export CLAUDE_CODE_ENABLE_OPUS_4_7_FAST_MODE=1
```

[Claude Code Fast Mode 문서](https://code.claude.com/docs/en/fast-mode)
